# playlist-app

- [x] Delete func
- [x] Modal
- [ ] Ability to add new items

- [ ] Search
- [ ] Infinite Scroll

**To try app on your local machine by downloading _Develop Branch!_**

### Project setup
```
npm install
```

### Compiles and minifies for production
```
npm run build
```

### Run app
```
node server/server.js
```

### And go to (http://localhost:4000/#/genres)
